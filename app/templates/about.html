About Us

This Book Management System is designed to help you organize and manage your book collection efficiently with advanced features including AI-powered recommendations.

Core Features:

  • Add, edit, and remove books from your collection
  • Track book details including title, author, ISBN, genre, and year
  • View and manage your complete book collection
  • Secure user authentication and account management

Advanced Features:

  • AI-Powered book recommendations based on your preferences
  • RESTful API access with Swagger documentation
  • Email notifications for account activities
  • Rate-limited API endpoints for security

Security Features:

  • Secure password hashing and validation
  • Protected routes requiring authentication
  • CSRF protection for forms
  • Rate limiting for API endpoints

Technical Specifications:

  • Built with Flask and SQLAlchemy
  • OpenAI GPT integration for recommendations
  • Celery for asynchronous task processing
  • Redis for task queue management
  • Flask-RESTX for API documentation
  • SQLite database for data storage

API Documentation:

Access our API documentation at API Docs (requires login) to explore available endpoints and features.
